# Tinypost

A tiny PostScript interpreter.

## Setup

Leiningen:

```clojure
[org.blancas/tinypost "0.1.0"]
```

Maven:

```xml
<dependency>
  <groupId>org.blancas</groupId>
  <artifactId>tinypost</artifactId>
  <version>0.1.0</version>
</dependency>
```
